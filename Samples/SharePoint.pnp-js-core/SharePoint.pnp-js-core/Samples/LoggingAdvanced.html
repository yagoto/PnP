Logging, Advanced

It is also possible to create your own log listener classes. This can be done easily as show by creating an object with a "log" method that takes a single parameter and subscribing that to the log. That single parameter will be a log message object having the following properties:
Property Type Description
message string The main message to be logged
level Verbose = 0, Info = 1, Warning = 2, Error = 3 The level of information this message represents
data string hash Any associated data that a given logging listener may choose to log or ignore
